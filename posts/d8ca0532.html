DSL回顾

快速回顾下这些Koin关键字：

  • module { } - 创建一个Koin模块或者子模块(在一个模块里面)
  • factory { } - 提供一个 factory工厂 bean定义
  • single { } - 提供一个bean定义
  • get() - 解析一个组件依赖
  • named() - 通过type、枚举或者字符串来定义一个修饰符
  • bind - 给给定的bean定义绑定其它Kolin类型
  • binds - 给给定的bean定义绑定其他Kotlin类型列表
  • getProperty() - 解析一个Koin属性
